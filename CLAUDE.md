## Project Overview

This is a full-stack todo application demonstrating Vue 3 + FastAPI integration. The frontend (Vue 3 + Tailwind CSS + Vite) communicates with a FastAPI backend through RESTful APIs.

## Architecture

### Frontend-Backend Communication
- Frontend runs on `http://localhost:5173` (Vite dev server)
- Backend runs on `http://localhost:8000` (FastAPI/Uvicorn)
- All API endpoints are prefixed with `/api`
- API_URL constant is hardcoded in `frontend/src/App.vue:107`
- CORS is configured in `backend/main.py:10-16` to allow both Vite (5173) and CRA (3000) ports

### Data Flow
- Backend uses in-memory storage (`todos_db` list in `backend/main.py:30`)
- No database - data is lost on server restart
- Frontend state management uses Vue 3 Composition API (`ref`, `computed`, `onMounted`)
- All data fetching happens in `App.vue` with async/await Fetch API calls

### API Endpoints (all in `backend/main.py`)
- `GET /api/todos` - Returns all todos
- `POST /api/todos` - Creates todo, expects `{title, description?}`
- `PUT /api/todos/{todo_id}` - Toggles completion status
- `DELETE /api/todos/{todo_id}` - Removes todo

### Data Models (`backend/main.py:19-27`)
- `TodoCreate`: Input validation (title required, description optional)
- `Todo`: Full model with id, title, description, completed status
- IDs are generated using `uuid.uuid4()`

## Development Commands

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py  # or: uvicorn main:app --reload
```

Backend API documentation available at `http://localhost:8000/docs` (auto-generated by FastAPI)

### Frontend
```bash
cd frontend
npm install
npm run dev       # Start development server
npm run build     # Production build
npm run preview   # Preview production build
```

## Key Implementation Details

### CORS Configuration
When adding new frontend ports or domains, update `allow_origins` in `backend/main.py:12`

### Frontend API Integration
The `API_URL` constant (`frontend/src/App.vue:107`) must match the backend URL. Update this if backend port changes.

### Styling
Tailwind CSS is configured via `tailwind.config.js` and processes files matching patterns in `content` array (`*.vue` files). PostCSS config is in `postcss.config.js`.

### Vue 3 Features
- Uses **Composition API** with `<script setup>` syntax
- Reactive state management with `ref` and `computed`
- Lifecycle hooks with `onMounted`
- Template directives: `v-model`, `v-if/v-else`, `v-for`, `@click`, `@submit`

## Technology Stack
- **Backend**: FastAPI, Uvicorn, Pydantic
- **Frontend**: Vue 3, Vite, Tailwind CSS
- **Language**: Python (backend), JavaScript (frontend)
